# End the World *(Parody)*

*The world's biggest opportunities. How to exploit them.*

A deadpan satirical parody — McKinsey deck for supervillains. **This is clearly labeled as parody.** Built with the [Eddie Design System](https://bradfrost.com).

## The Point

This site is one half of a pair. Its sibling — [Save the World](../ultimate-good/) — uses the **exact same design system**, same components, same layout patterns. The only difference is one accent color token and the content.

Design systems don't have values. People do.

## Running Locally

```bash
npm install
npm run dev
```

## Building

```bash
npm run build
npm run preview
```

## Tech Stack

- Vanilla HTML + Eddie Design System web components
- Vite for dev server and production build
- No framework — static HTML pages
- Eddie Design Tokens (BFW theme) with crimson accent override
